# rust-compile-over-network
allows you to upload your rust projects over ssh, compile them on a remote server over ssh, and downloads it OVER SSH :O
this is in a "it just works with no extra features" state
I think nothing needs to be explained except that there is no support (right now) to get a release build, 
and it just copies over the compiled executable to ~/sshnetcomp
also it doesn't check for the proper compilers so uh yeah enjoy compiling or something
